Inorder traversal

(with python)

In the following demonstration, I will guide you through an inorder traversal of a binary tree using the tree structure depicted in the image above.

When tackling traversal problems like this, I find it helpful to work with three variables: one for the result, one to hold the current stack, and one for the current node being traversed.

In this specific problem, a stack is essential because in many cases, you'll need to traverse deeper down the node before you can pop it to the result. Therefore, a stack provides temporary storage for this process.

Now that we have our three variables, we can traverse the tree. Our first loop will continue until there is no current node or stack, which indicates that the tree has been fully traversed. Within the loop, since it's an inorder traversal, we'll always append the current node to the stack and then move to the left, continuing until we reach the end. At that point, we append the leftmost node to the result, move to the right node (if available), and restart the process.
