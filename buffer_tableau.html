Buffers in Tableau

(with New York datasets)

Research question

     Are EV ports close to AirBnb's in NYC?


To answer our research question, we will utilize two datasets: one detailing the locations of all the AirBnb's in NYC and another indicating the locations of all the EV ports in NYC.


Download New York AirBnb dataset

Download New York EV ports dataset

AirBnb points

First, let's create visual points for all the AirBnb's in NYC. We can achieve this using the makepoint function in Tableau to transform the latitude and longitude variables into visual points.




EV points

We can add the EV data to our view and follow the same process to obtain all the EV charging points. Once we have the EV points, we can incorporate them into our view as a new layer and modify their shape and color to ensure clear visibility.




Joining two datasets

We can observe the locations of our AirBnb's and EV ports on the map. However, to determine the distance between them, we need to merge the two datasets. Below, you can see how I employed an inner join to achieve this.




Distance function

Now, we will need to utilize the Distance function to create a variable that calculates the distance between an AirBnb and EV ports.




Buffer function

We will employ the buffer function to create a buffer circle around the EV ports, displaying the distance of the buffer circle as the radius.




finishing touches

Lastly, we can adjust the color and transparency of our buffer circle to indicate a radius of 150 meters. Additionally, we can modify the distance value from SUM, which currently encompasses all EV ports in its buffer circle, to calculate the distance only to the nearest EV port.




Conclusion

It's evident that, for the most part, there is an EV port within at least 150 meters of every AirBnb in downtown Manhattan. To extend our analysis to cover the entirety of NYC, we can create additional buffer circles in any area of interest.
